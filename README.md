# Claude Code for Everyone - Kursuse materjalid

Tere tulemast **Claude Code for Everyone** (CC4E) kursusele - praktilisele kursusele, mis õpetab Claude Code'i kasutamist mittetehnilistele kasutajatele.

## Alustamine

1. Ava see kaust Cursoris (või VS Code'is)
2. Ava terminal (Ctrl+` või View → Terminal)
3. Kirjuta `claude` ja vajuta Enter
4. Kirjuta `/start-1-1` alustamiseks!

## Kursuse struktuur

### Moodul 1: Claude Code'i põhitõed
- **1.1** Sissejuhatus - Tervitus ja kursuse ülevaade
- **1.2** Failide uurimine - Navigeeri failides ja seadista oma tööruum
- **1.3** Failidega töötamine - Väljavõtted, süntees, mallid, pildid, veeb
- **1.4** Käsud ja navigeerimine - Otseteed ja edasijõudnute funktsioonid
- **1.5** Agendid - Paralleeltöötlus mitme Claude'i instantsiga
- **1.6** Kohandatud alamagendid - Loo oma nõuandjate meeskond
- **1.7** CLAUDE.md - Püsiv projektimälu
- **1.8** Võimsamad funktsioonid - Tähistamine ja Mooduli 2 eelvaade

### Moodul 2: Vibe Coding (Varsti tulemas)
Ehita Kohvi Isiksuse Viktoriin nullist!

## Stsenaarium

Sa oled Basecamp Coffee uus juht - piirkondlik kohvikett, mille lojaalsusprogramm lonkab. Sinu ülesanne: pööra see 3 kuuga.

Kursuse jooksul sa:
- Analüüsid kaost, mille sa pärisid
- Avastad, mis programmiga valesti on
- Leiad lahenduse (spoiler: see hõlmab kohviisiksusi)
- Ehitad prototüübi, mis päästab päeva

## Kaustade struktuur

```
company-context/     → Taustainfo Basecamp Coffee kohta
inherited-chaos/     → Segadus, mille sa pärisid (failid analüüsimiseks)
templates/           → Dokumendimallid
attachments/         → Pildid ja muud materjalid
organized/           → Failid, mida lood korraldades
analysis/            → Analüüsi väljundid, mida genereerid
reviews/             → Tagasiside alamagentidelt
```
